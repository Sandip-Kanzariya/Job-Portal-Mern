# Job-Portal Using MERN

### Run Project 

```
go inside job-portal

cd job-portal
npm install 

cd frontend 
npm install
npm start 

nodemon index.js / node index.js 
```

### .env file for

```
NODE_ENV = development / production
PORT = 4500
DATABASE = 
```

### Dependencies

```
npm i bcryptjs dotenv express mongoose
```

### Dev Dependencies

```
npm i -D nodemon
```
